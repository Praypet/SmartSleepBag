### 1. 获取宝宝信息

获取当前宝宝的个人信息。

**接口**: `getBabyInfo`  
**方法**: GET  
**路径**: `/home/baby-info`

#### 请求参数

无

  "code": 200,

  "message": "string"

## 错误码说明

| 错误码 | 描述 | 处理建议 |
|:------:|------|----------|
| 200 | 成功 | - |
| 400 | 请求参数错误 | 检查请求参数格式 |
| 401 | 未授权 | 重新登录获取token |
| 403 | 禁止访问 | 检查用户权限 |
| 404 | 资源不存在 | 检查请求路径 |
| 429 | 请求过于频繁 | 稍后重试 |
| 500 | 服务器内部错误 | 联系管理员 |
| 503 | 传感器离线 | 检查设备连接状态 |
